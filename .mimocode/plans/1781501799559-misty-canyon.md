| Area | Status | Key Issues |

## PHASE 1: Design System Foundation (Day 1-2)

### 1.1 Standardize Color Tokens
- **File**: `tailwind.config.js` вҖ” Add semantic tokens, remove orphaned `card`/`button` radius
- **File**: `app/globals.css` вҖ” Sync CSS custom properties with Tailwind config

```js
// tailwind.config.js additions
colors: {
  surface: { DEFAULT: '#F8F8F8', dark: '#F0F0F0' },
  border: { DEFAULT: '#E8E8E8' },
  text: { primary: '#111111', secondary: '#555555', muted: '#999999' },
  // Keep existing: primary (black), accent (#25D366)
}
```

**Fix**: Replace all ad-hoc hex values across codebase:
- `bg-[#F9FAFB]` вҶ’ `bg-surface`
- `text-[#555]` вҶ’ `text-text-secondary`
- `border-[#F0F0F0]` вҶ’ `border-border`
- `text-[#999]` вҶ’ `text-text-muted` (and darken to `#888` for contrast)

### 1.2 Fix Broken Animations
- **File**: `app/globals.css` вҖ” Add missing keyframes:

